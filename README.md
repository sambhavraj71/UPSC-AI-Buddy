# UPSC AI Buddy - Smart Study Companion

An AI-powered website to help UPSC aspirants with intelligent study assistance using Google Gemini API.

## Features

- 🤖 AI Study Buddy Chat
- 📝 Answer Evaluation
- 🎯 Smart Quiz Generator
- 📊 Progress Tracking
- 📚 Study Plan Generator
- 🔥 Streak Counter
